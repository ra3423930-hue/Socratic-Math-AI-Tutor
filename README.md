A Python-based AI assistant prompted to act as a strict math teacher using LLM APIs.
# 🎓 Chandigarh Engineering Counselling & Academic Assistant

An automated, terminal-based AI assistant built during my pre-college break to bridge the gap between competitive mathematics (IOQM level) and applied engineering logic.

## 🚀 Overview
This project utilizes Google's latest Gemini 2.5 Flash SDK to initialize a dynamic, dual-mode system prompt. It acts as an interactive mentor while I navigate engineering counselling and prepare for my upcoming B.Tech journey.

## 🔧 Features
- **Counselling Mode:** Evaluates college networks, branch upgrades, and academic paths objectively.
- **Socratic Academic Tutor:** Evaluates complex topics in programming, calculus, and circuit physics using a strict Socratic teaching method (asking guiding questions instead of giving immediate answers).

## 🛠️ Tech Stack
- **Language:** Python 3
- **LLM Engine:** Google Gemini 2.5 Flash API (`google-genai` SDK)

## 📦 How to Run
1. Install dependencies:
   ```bash
   pip install google-genai
   ```
2. Run the application:
   ```bash
   python tutor.py
   ```
